ℹ️ À propos de Daily Home Kid Challenge

Daily Home Kid Challenge est une application éducative ludique qui aide les enfants à suivre leurs tâches quotidiennes et à recevoir des récompenses motivantes basées sur leur engagement.

⭐ Fonctionnalités principales

  • 👨‍👩‍👧 Gestion multi-enfants avec paramètres personnalisés
  • 📝 Tableau quotidien / hebdomadaire / mensuel avec cases à cocher
  • ⚖️ Pondération des tâches par jour
  • 📊 Calcul automatique du pourcentage de réussite hebdomadaire
  • 🎁 Récompenses personnalisées ou par défaut avec paliers configurables
  • 📖 Historique avec tableau et graphique d’évolution
  • ♻️ Réinitialisation (par semaine, par enfant ou totale)
  • 📥 Import / 📤 Export d’un enfant (JSON)
  • 📱 PWA : fonctionne hors ligne et installable sur mobile / PC
  • 🌙 Mode clair / sombre

🎁 Règles de récompense

  • Moins que le seuil configuré → ❌ Aucune récompense
  • Entre Palier 1 et Palier 2 → 🎁 Récompense Palier 1
  • À partir de Palier 2 → 🏆 Récompense Palier 2

🔒 Données & confidentialité

Toutes les données sont stockées localement dans ton navigateur (LocalStorage). Rien n’est envoyé sur un serveur externe.

📘 Guide d’utilisation

  • ➕ Ajouter un enfant : crée un nouvel enfant avec ses paramètres.
  • 💾 Enregistrer : sauvegarde manuelle (en plus de l’auto-save).
  • 🔄 Réinitialiser enfants : supprime tous les enfants.
  • 💣 Purge totale : supprime toutes les données (protégée par confirmation).
