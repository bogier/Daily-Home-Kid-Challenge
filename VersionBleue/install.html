📲 Installer l’application

Ajoute Daily Home Kid Challenge sur ton écran d’accueil pour profiter de l’expérience complète, même hors ligne !

🍎🌐

iPhone / iPad (Safari)

  1. Ouvre le site dans Safari.
  2. Appuie sur Partager (icône carré + flèche).
  3. Choisis Ajouter à l’écran d’accueil.
  4. Valide avec Ajouter.

💡 Astuce : si l’option n’apparaît pas, fais défiler la fiche de partage vers le bas.

🤖🌈

Android (Chrome)

  1. Ouvre le site dans Chrome.
  2. Appuie sur ⋮ en haut à droite.
  3. Choisis Installer l’application ou Ajouter à l’écran d’accueil.
  4. Confirme l’installation.
💻✨

Ordinateur (Chrome / Edge)

  1. Ouvre le site dans Chrome ou Edge.
  2. Clique sur l’icône Installer dans la barre d’adresse.
  3. Confirme avec Installer.
